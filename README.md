# Danse Macabre

## Compile

	# make

## Run

Linux, OS X	

	# ./DanseMacabre

Windows

	# DanseMacabre.exe


## Control keys

- a, z - camera horizontal angle
- s, x - camera distance
- d, c - camera altitude
- f, v - lights altitude
- q - drawing mode (triangles, lines, points)
- w - automatic camera rotation
